Accessibility

Last updated: November 2025

My commitment

I want this site to be usable and understandable for as many people as possible, including people using assistive technologies such as screen readers, keyboard-only navigation, or high-contrast modes.

What I’ve done so far

  • Used simple layouts and clear headings for content structure.
  • Included a “Skip to main content” link at the top of each page.
  • Avoided auto-playing audio and heavy motion; soft animations respect reduced-motion settings.
  • Used sufficient color contrast for text and key controls where possible.
  • Ensured main actions (booking, contact, purchase) are accessible via keyboard navigation.

Where there may still be issues

Some parts of the site rely on third-party widgets, such as:

  • Calendly for scheduling.
  • Stripe Buy Buttons for payments.
  • HubSpot forms for contact submissions.

These services control their own markup and behavior, and while they generally support accessibility, there may be limitations or differences compared to the rest of the site.

How to report a problem

If you experience any issues using this site, especially with assistive technology or keyboard navigation, please let me know:

  • Email: plewak.jeff@gmail.com
  • Subject line: “Accessibility feedback” if you’re comfortable using that phrase.

If you can include details like the page, what you were trying to do, and the tools you use (for example screen reader or browser), that helps me fix the problem faster.

Ongoing improvements

I will continue to review and refine the site, especially when adding new sections or tools, to keep it as clear and accessible as I reasonably can.
