2) Next optimization: SEO + lead capture (no more UI rewrites)

You’re now at the point where tech is “good enough” and the bottleneck becomes traffic and follow-up. Without asking you anything, here’s what I’d do next, concretely:

A. Turn jeffreyplewak.com into the canonical hub

Use one link everywhere:

https://jeffreyplewak.com/

Every profile (LinkedIn, GitHub, Upwork, Fiverr, Reddit, X, etc.) should point here, not to subpages.

Then let the site route people:
	•	“Book consultation” → /book.html
	•	“See pricing” → /pay.html
	•	“Just send a message” → /contact.html

B. Use simple UTM tags for attribution

When you set profile links, use clean UTM strings so HubSpot sees source properly. Examples:
	•	LinkedIn:

https://jeffreyplewak.com/?utm_source=linkedin&utm_medium=profile&utm_campaign=personal_brand


	•	Reddit:

https://jeffreyplewak.com/?utm_source=reddit&utm_medium=profile&utm_campaign=personal_brand


	•	Upwork/Fiverr:

https://jeffreyplewak.com/?utm_source=upwork&utm_medium=profile&utm_campaign=freelance



HubSpot will pick those up automatically on contact submissions.

C. HubSpot: minimum viable CRM setup

Inside HubSpot (Free):
	1.	Create a simple pipeline / board:
	•	Columns like: New → Qualified → Call booked → Proposal → Active → Won / Lost.
	2.	Create a saved view:
	•	Filter: contacts where Original source drill-down includes your site or socials.
	3.	Set a task rule for new contacts:
	•	Whenever a new contact comes from the website, create a follow-up task “Review & reply” due next day.

This way, every form submission from contact.html becomes a task you can’t ignore.
